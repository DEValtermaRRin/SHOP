# E-shop #

## Описание: ##

Данный проект являтся учебным
В нем используется:
* HTML 5
* CSS 3
    * @import
    * @media
    * семантические теги
    * псевдоклассы и псевдоэлементы
    * небольшие эффекты анимации
    * методология БЭМ
* SASS
    * @mixin
* JS
    * работа с корзиной
