# E-shop #

## Описание: ##

Данный проект являтся учебным и представляет адаптивную верстку интернет-магазина одежды. Логики почти никакой нет

В проекте используется:
* HTML 5
* CSS 3
    * @import
    * @media
    * семантические теги
    * псевдоклассы и псевдоэлементы
    * небольшие эффекты анимации
    * методология БЭМ
* SASS
    * @mixin
    * $vars
* JS
    * работа с корзиной
